This is a fast paced arithmetic game, inspired from the website - ZetaMac.
Basically - you get to choose a duration and mathematical operations, and in that time, you need to solve as many calculations as possible.

To contribute - 
 - clone the repository
 - run the code.
 - No other setup is required. 

 Java 25 is preferred.

 To play:
 Go to the releases and download the .exe file (a warning will be issued, but it is safe to install)
 Go to the cmd and type this: "C:\Program Files\Zetamac\Zetamac.exe"   (with the double quotes)
